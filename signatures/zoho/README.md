## How to add one in Zoho Mail

1. Open the needed `.html` file in a browser.
2. Select the visible signature card and copy it.
3. In Zoho Mail, go to **Settings > Signatures**.
4. Create or edit a signature and paste the copied signature into the editor.
5. Save it and send yourself a test email.

If Zoho does not preserve the design when copying from the browser, open the `.html` file in a text editor, copy the HTML source, and paste it into Zoho's HTML/source editor if available.

## How to send these on Telegram

Desktop is easiest:

1. Open Telegram Desktop or Telegram Web.
2. Open the chat where you want to send the signatures.
3. Click the attachment/paperclip button.
4. Choose **File**, then select the `.html` files from `signatures/zoho`.
5. Send them as files, not as pasted text, so the HTML stays intact.

You can also zip the folder first and send one file:

```bash
zip -r zoho-signatures.zip signatures/zoho
```
